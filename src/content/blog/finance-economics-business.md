## Sección 1: Fundamentos de Finanzas

### Conceptos Clave

#### ROI (Return on Investment)

Mide el rendimiento de una inversión.

```
ROI = (Ganancia - Inversión) / Inversión × 100

Ejemplo:
Inviertes $100, ganas $120
ROI = ($120 - $100) / $100 × 100 = 20%
```

#### Valor Temporal del Dinero

> **💡 Concepto Clave**: Un peso hoy vale más que un peso mañana.

**Por qué**: Porque el dinero disponible hoy puede invertirse y generar rendimientos.

#### VPN (Valor Presente Net / NPV)

Calcula el valor actual de flujos futuros descontados.

```
VPN = Σ (Flujo / (1 + tasa)^n) - Inversión inicial

Ejemplo:
Flujo año 1: $50, Año 2: $50, Año 3: $50
Tasa descuento: 10%
Inversión: $100

VPN < 0 → Proyecto no rentable
VPN > 0 → Proyecto rentable
```

## Sección 2: Mercados Financieros

### Acciones (Stocks)

```
┌─────────────────────────────────────────┐
│              ACCIÓN                     │
├─────────────────────────────────────────┤
│  Ownership (propiedad) de empresa     │
│  derecho a Dividendos                 │
│  Voto en asambleas                    │
│  Ganancias de capital (precio sube)    │
│  Riesgo: ilimitado (puede perder todo) │
└─────────────────────────────────────────┘
```

### Bonos (Bonds)

```
┌─────────────────────────────────────────┐
│              BONO                       │
├─────────────────────────────────────────┤
│  Préstamo a empresa/gobierno           │
│  Interés fijo (cupón)                  │
│  devolução a vencimiento              │
│  Menor riesgo que acciones            │
│  Income estable pero sin crecimiento  │
└─────────────────────────────────────────┘
```

### Acciones vs Bonos

| Característica | Acciones  | Bonos |
| -------------- | --------- | ----- |
| **Riesgo**     | Alto      | Bajo  |
| **Retorno**    | Ilimitado | Fijo  |
| **Voto**       | Sí        | No    |
| **Dividendos** | Variable  | Fijo  |

## Sección 3: Estrategia Empresarial

### Declaración de Misión

> **🔍 Para Curiosos**: La misión define el "para qué" de la empresa.

```
Ejemplo de misión bien formulada:
"Democratizar el acceso a servicios financieros
para millones de personas no bancarizadas"
```

### Análisis FODA (SWOT)

```
              │ FORTALEZAS (S)    │ DEBILIDADES (W)
              │------------------│------------------
   INTERNO    │ • Marca fuerte   │ • Costes altos
              │ • Capital        │ • Dependencia
              │ • Tecnología     │ • Equipo pequeño
──────────────┼──────────────────┼───────────────────
              │ OPORTUNIDADES(O) │ AMENAZAS (T)
   EXTERNO     │------------------│------------------
              │ • Nuevo mercado │ • Competencia
              │ • Regulation    │ • Tecnológica
              │ • Alianzas      │ • Económicas
```

### Matriz BCG (Boston Consulting Group)

```
                    Tasa de crecimiento
                           ↑
                    Alta │    Estrella   │    Incógnita
                         │                │
        ──────────────────┼────────────────┼──→
                    Baja │     VACA      │    PERRO
                         │   (cash cow)  │
                           │
                    Bajo │───────────────│ Alta
                    Cuota de mercado →
```

| Cuadrante     | Descripción                   | Acción     |
| ------------- | ----------------------------- | ---------- |
| **Estrella**  | Alto crecimiento, alto market | Invertir   |
| **Incógnita** | Bajo crecimiento, alto market | Evaluar    |
| **Vaca**      | Bajo crecimiento, bajo market | Extraer    |
| **Perro**     | Bajo crecimiento, bajo market | Deshacerse |

## Sección 4: Estados Financieros

### Estado de Resultados (Income Statement)

```
┌─────────────────────────────────────────────────┐
│         ESTADO DE RESULTADOS                    │
│         Período: Año 2024                       │
├─────────────────────────────────────────────────┤
│ (+) Ventas                      $1,000,000     │
│ (-) Costo de Ventas              ($400,000)     │
├─────────────────────────────────────────────────┤
│ = Utilidad Bruta                 $600,000       │
│ (-) Gastos Operativos          ($250,000)      │
│ (-) Depreciación               ($50,000)       │
├─────────────────────────────────────────────────┤
│ = Utilidad Operativa (EBIT)      $300,000       │
│ (-) Intereses                 ($30,000)       │
│ (-) Impuestos                 ($70,000)       │
├─────────────────────────────────────────────────┤
│ = Utilidad Neta                 $200,000       │
└─────────────────────────────────────────────────┘
```

### Balance General (Statement of Financial Position)

```
┌─────────────────────────────────────────────────┐
│              BALANCE GENERAL                    │
│               Año 2024                         │
├─────────────────┬─────────────────────────────┤
│    ACTIVOS      │      PASIVOS                 │
├────────────────┼─────────────────────────────┤
│ Caja      $50K  │ Proveedores        $30K     │
│ Cuentas  $80K  │ Deudas bancarias  $150K    │
│ Inventario $70K│               │
│ Activos   $200K │       TOTAL PASIVOS $180K   │
│                 │                              │
├────────────────┼─────────────────────────────┤
│                 │   PATRIMONIO                │
│                 ├─────────────────────────────┤
│                 │ Capital social    $100K     │
│                 │ Utilidades acum. $120K     │
│                 │       PATRIMONIO $220K     │
├────────────────┼─────────────────────────────┤
│ TOTAL $400K    │ TOTAL P+P $400K            │
└────────────────┴─────────────────────────────┘
```

### Flujo de Caja (Cash Flow)

```
┌─────────────────────────────────────────────────┐
│               FLUJO DE CAJA                    │
├─────────────────────────────────────────────────┤
│ FLUJO OPERATIVO                                │
│ + Utilidad Neta              $200,000           │
│ + Depreciación                $50,000           │
│ - Aumento en cuentas por cobrar ($20,000)     │
│ = Flujo de operaciones          $230,000       │
├─────────────────────────────────────────────────┤
│ FLUJO DE INVERSIÓN                             │
│ - Compra de equipo            ($100,000)       │
│ = Flujo de inversión           ($100,000)       │
├─────────────────────────────────────────────────┤
│ FLUJO DE FINANCIAMIENTO                        │
│ + Nuevo préstamo            $50,000            │
│ - Pago de dividendos        ($30,000)         │
│ = Flujo de financiamiento      $20,000           │
├─────────────────────────────────────────────────┤
│ Variación neta de caja    $150,000            │
│ + Caja inicial                $50,000         │
│ = Caja final                 $200,000         │
└─────────────────────────────────────────────────┘
```

## Sección 5: Técnicas de Análisis Financiero

### Ratios Financieros

| Ratio             | Fórmula                    | Interpretación          |
| ----------------- | -------------------------- | ----------------------- |
| **Margen Neto**   | Utilidad/Ventas × 100      | % de ganancia           |
| **ROE**           | Utilidad/Patrimonio        | Retorno a accionistas   |
| **ROA**           | Utilidad/Activos           | Eficiencia de activos   |
| **Actual**        | Activos/Pasivos corrientes | Liquidez inmediata      |
| **Rapido**        | (Activos-Inv)/Pasivos      | Liquidez sin inventario |
| **Endeudamiento** | Pasivos/Patrimonio         | Apalancamiento          |

### Análisis Horizontal (Tendencias)

```
       2022    2023    2024    Tendencia
Ventas $800K  $950K  $1.1M  +37.5%

Analiza: Crecimiento año a año
```

### Análisis de Tamaño Común

```
                    2023      2024
Ventas             $950K     70%    100%
Costo              ($380K)   40%    42%
Utilidad Bruta     $570K     60%    58%

Compara percentages en lugar de valores absolutos
```

## Sección 6: Capital Budgeting

### TIR (Internal Rate of Return)

La tasa que hace el VPN igual a cero.

```
TIR > Costo de Capital → Aceptar proyecto
TIR < Costo de Capital → Rechazar proyecto
```

### Comparación de Proyectos

| Proyecto | Inversión | TIR | VPN  |
| -------- | --------- | --- | ---- |
| A        | $100K     | 15% | $20K |
| B        | $80K      | 18% | $18K |
| C        | $150K     | 12% | $25K |

**Selección**: Enfoque práctico → Proyecto A y B son atractivos.

## Sección 7: Macroeconomía

### Ciclo Económico

```
            Expansión          Recesión
               │                  │
      Pico ────┼──── Valle         │
               │                  │
          ↑↑↑↑↑↑↑↑↑           ↓↓↓↓↓↓
          Peak                Trough
                    △ Recesión

           Recovery    Expansion    Peak    Recession
```

### Política Fiscal

| Política        | Herramienta        | Efecto            |
| --------------- | ------------------ | ----------------- |
| **Expansiva**   | ↑Gastos -Impuestos | Estimula economía |
| **Contractiva** | ↓Gastos +Impuestos | Enfrena inflación |

### Política Monetaria

| Política        | Herramienta | Efecto          |
| --------------- | ----------- | --------------- |
| **Expansiva**   | Bajar tasas | Más crédito     |
| **Contractiva** | Subir tasas | Menos inflación |

> **⚠️ Importante**: Las tasas altas atraen inversión extranjera.

## Sección 8: ESG y Sostenibilidad

### Qué Es ESG

| Letra | Significado   | Ejemplo                       |
| ----- | ------------- | ----------------------------- |
| **E** | Environmental | Emisiones, residuos           |
| **S** | Social        | Derechos laborales, comunidad |
| **G** | Governance    | Corrupción,董事会 diversidad  |

### Por Qué Importa ESG

```
Antes: Solo ROI importaba
Ahora: ESG afecta:
- Acceso a capital
- Reputación de marca
- Regulaciones
- Younger consumers
```

## Sección 9: Gestión de Portafolio

### Diversificación

> **💡 Concepto Clave**: "No pongas todos los huevos en la misma canasta".

```
Cartera bien diversificada:
├── 40% Bonos (estabilidad)
├── 30% Acciones crecimiento
├── 20% Acciones valor
└── 10% Alternativos (real estate, commodities)
```

### Medidas de Rendimiento

| Métrica    | Descripción                        |
| ---------- | ---------------------------------- |
| **Alpha**  | Rendimiento excedente vs benchmark |
| **Beta**   | Sensibilidad al mercado            |
| **Sharpe** | Retorno ajustado por riesgo        |

### Tipos de Inversión Alternativa

| Tipo               | Característica           |
| ------------------ | ------------------------ |
| **Real Estate**    | Bienes raíces            |
| **Commodities**    | Materias primas          |
| **Private Equity** | Empresas privadas        |
| **Hedge Funds**    | Estrategias alternativas |

## Errores Comunes

### ❌ Error 1: "Ignorar el balance"

**Mejor**: Siempre balance = activo + pasivo + patrimonio cuadra.

### ❌ Error 2: "No analisar tendencias"

**Mejor**: Un año aisladamente no dice nada.

### ❌ Error 3: "Confundir cash flow con utilidad"

**Mejor**: Una empresa puede ser rentable sin caja.

### ❌ Error 4: "Ignorar ESG"

**Mejor**: ESG cada vez más relevante.

### ❌ Error 5: "No diversificar"

**Mejor**: Cartera diversificada reduce riesgo.
