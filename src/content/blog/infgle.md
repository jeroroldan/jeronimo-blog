## 1. LOS 3 TIEMPOS QUE CUBREN EL 90% DE LAS CONVERSACIONES {#tiempos}

### ⏱️ PRESENTE SIMPLE — Lo que hacés habitualmente

**Estructura:**

```
I / You / We / They  +  verbo base
He / She / It        +  verbo + S
```

| Español     | Inglés   |
| ----------- | -------- |
| Yo trabajo  | I work   |
| Él trabaja  | He works |
| Ellos comen | They eat |
| Ella come   | She eats |

**Negación:** sujeto + **don't / doesn't** + verbo

- I don't eat meat. _(No como carne)_
- She doesn't like coffee. _(A ella no le gusta el café)_

**Pregunta:** **Do / Does** + sujeto + verbo?

- Do you speak English? _(¿Hablás inglés?)_
- Does he live here? _(¿Él vive acá?)_

---

### ⏱️ PASADO SIMPLE — Lo que ya pasó

**Estructura:**

```
sujeto + verbo en pasado (verbos regulares: + ED)
```

| Presente | Pasado   | Español |
| -------- | -------- | ------- |
| work     | worked   | trabajé |
| play     | played   | jugué   |
| go       | **went** | fui     |
| eat      | **ate**  | comí    |
| have     | **had**  | tuve    |

> ⚠️ Los verbos irregulares hay que memorizar, pero los 20 más comunes te cubren el 80% de las situaciones.

**Negación:** sujeto + **didn't** + verbo base

- I didn't go. _(No fui)_
- She didn't eat. _(Ella no comió)_

**Pregunta:** **Did** + sujeto + verbo base?

- Did you call me? _(¿Me llamaste?)_
- Did he arrive? _(¿Él llegó?)_

---

### ⏱️ FUTURO CON "GOING TO" — Planes e intenciones

**Estructura:**

```
sujeto + am/is/are + going to + verbo base
```

| Español          | Inglés               |
| ---------------- | -------------------- |
| Voy a trabajar   | I am going to work   |
| Ella va a llamar | She is going to call |
| Vamos a comer    | We are going to eat  |

**Negación:** sujeto + am/is/are + **not** + going to + verbo

- I am not going to go. _(No voy a ir)_

**Pregunta:** **Am/Is/Are** + sujeto + going to + verbo?

- Are you going to come? _(¿Vas a venir?)_

---

### 🔑 VERBO "TO BE" — El más importante de todos

| Español         | Afirmativo    | Negativo    | Pregunta  |
| --------------- | ------------- | ----------- | --------- |
| Yo soy/estoy    | I **am**      | I'm not     | Am I?     |
| Vos sos/estás   | You **are**   | You aren't  | Are you?  |
| Él/ella es/está | He/She **is** | He isn't    | Is he?    |
| Nosotros somos  | We **are**    | We aren't   | Are we?   |
| Ellos son       | They **are**  | They aren't | Are they? |

**Ejemplos:**

- I am tired. _(Estoy cansado)_
- She is a doctor. _(Ella es doctora)_
- We are from Argentina. _(Somos de Argentina)_
- Are you okay? _(¿Estás bien?)_

## 3. FÓRMULAS MÁGICAS PARA EMPEZAR CUALQUIER ORACIÓN {#formulas}

> Esta es la sección más importante. Si sabés cómo **arrancar** una oración, el resto fluye solo.

### 🟢 Para hablar de VOS MISMO

| Fórmula                | Ejemplo                        |
| ---------------------- | ------------------------------ |
| **I am...**            | I am from Argentina.           |
| **I work...**          | I work in Buenos Aires.        |
| **I like...**          | I like music.                  |
| **I don't...**         | I don't eat meat.              |
| **I want to...**       | I want to travel.              |
| **I need to...**       | I need to learn English.       |
| **I think...**         | I think it's a good idea.      |
| **I know...**          | I know that place.             |
| **I don't know...**    | I don't know the answer.       |
| **I would like...**    | I would like a coffee, please. |
| **I'm looking for...** | I'm looking for the station.   |
| **I have...**          | I have two brothers.           |
| **I used to...**       | I used to play football.       |

### 🔵 Para hablar DE OTROS

| Fórmula                | Ejemplo                 |
| ---------------------- | ----------------------- |
| **He/She is...**       | She is my sister.       |
| **They are...**        | They are my friends.    |
| **He/She works...**    | He works in a hospital. |
| **His/Her name is...** | Her name is Laura.      |

### 🟡 Para hacer PREGUNTAS

| Fórmula                  | Ejemplo                        |
| ------------------------ | ------------------------------ |
| **Can you...?**          | Can you help me?               |
| **Do you...?**           | Do you speak English?          |
| **Are you...?**          | Are you okay?                  |
| **Is there...?**         | Is there a bathroom nearby?    |
| **Where is...?**         | Where is the hotel?            |
| **How much is...?**      | How much is this?              |
| **What time is...?**     | What time is it?               |
| **What does ... mean?**  | What does "fierce" mean?       |
| **How do you say...?**   | How do you say "lluvia"?       |
| **Could you repeat...?** | Could you repeat that, please? |

### 🟠 Para dar OPINIONES o HABLAR DE PLANES

| Fórmula                 | Ejemplo                       |
| ----------------------- | ----------------------------- |
| **I think that...**     | I think that it's expensive.  |
| **In my opinion...**    | In my opinion, this is wrong. |
| **I'm going to...**     | I'm going to call you later.  |
| **I'm planning to...**  | I'm planning to visit London. |
| **Maybe I will...**     | Maybe I will try.             |
| **I hope to...**        | I hope to see you soon.       |
| **I'm not sure but...** | I'm not sure but I think so.  |

### 🔴 Para cuando NO ENTENDÉS algo

| Fórmula                          | Ejemplo                         |
| -------------------------------- | ------------------------------- |
| **Sorry, could you...?**         | Sorry, could you speak slowly?  |
| **I don't understand...**        | I don't understand this word.   |
| **What do you mean by...?**      | What do you mean by that?       |
| **Can you explain...?**          | Can you explain this to me?     |
| **I'm sorry, I didn't catch...** | I'm sorry, I didn't catch that. |

## 4. PREGUNTAS ESENCIALES {#preguntas}

### Las 6 preguntas base (WH Questions)

| Pregunta   | Uso      | Ejemplo             |
| ---------- | -------- | ------------------- |
| **What?**  | Qué/Cuál | What is your name?  |
| **Where?** | Dónde    | Where are you from? |
| **When?**  | Cuándo   | When does it start? |
| **Who?**   | Quién    | Who is that?        |
| **Why?**   | Por qué  | Why are you late?   |
| **How?**   | Cómo     | How are you?        |

### Variaciones de HOW (las más usadas)

| Pregunta   | Significado          | Ejemplo                    |
| ---------- | -------------------- | -------------------------- |
| How much?  | ¿Cuánto? (precio)    | How much is it?            |
| How many?  | ¿Cuántos? (cantidad) | How many people?           |
| How long?  | ¿Cuánto tiempo?      | How long does it take?     |
| How far?   | ¿Qué tan lejos?      | How far is it?             |
| How often? | ¿Con qué frecuencia? | How often do you exercise? |
| How old?   | ¿Qué edad?           | How old are you?           |

## 5. FRASES DE SUPERVIVENCIA SOCIAL {#supervivencia}

### Saludos y presentaciones

```
Hello! / Hi!               → ¡Hola!
Good morning!              → ¡Buenos días!
Good afternoon!            → ¡Buenas tardes!
Good evening!              → ¡Buenas noches! (al llegar)
How are you?               → ¿Cómo estás?
I'm fine, thanks. And you? → Bien, gracias. ¿Y vos?
Nice to meet you!          → ¡Mucho gusto!
My name is...              → Mi nombre es...
I'm from Argentina.        → Soy de Argentina.
```

### En una tienda o restaurante

```
I'd like...                → Quisiera...
Can I have...?             → ¿Me puede dar...?
How much is this?          → ¿Cuánto cuesta esto?
Do you accept cards?       → ¿Aceptan tarjetas?
The check, please.         → La cuenta, por favor.
What do you recommend?     → ¿Qué recomendás?
```

### Pidiendo ayuda o información

```
Excuse me...               → Disculpe...
Where is...?               → ¿Dónde está...?
How do I get to...?        → ¿Cómo llego a...?
Is it far?                 → ¿Está lejos?
Can you help me?           → ¿Podés ayudarme?
I'm lost.                  → Estoy perdido.
```

### Despedidas

```
Goodbye! / Bye!            → ¡Adiós! / ¡Chau!
See you later!             → ¡Hasta luego!
See you tomorrow!          → ¡Hasta mañana!
Take care!                 → ¡Cuidate!
Have a nice day!           → ¡Que tengas un buen día!
It was nice talking to you. → Fue un gusto hablar con vos.
```

## 6. TIPS Y TRUCOS DE UN EXPERTO {#tips}

### 🧠 El truco del "VERB + ING" para sonar natural

Cuando querés describir lo que **estás haciendo ahora mismo**, usá `am/is/are + verbo-ING`:

- I am **working**. _(Estoy trabajando)_
- She is **sleeping**. _(Ella está durmiendo)_
- We are **waiting**. _(Estamos esperando)_

Este tiempo (Present Continuous) también sirve para planes inmediatos:

- I am **leaving** tomorrow. _(Me voy mañana)_

---

### 🎯 El truco del "I WOULD LIKE" para ser educado siempre

En vez de decir **"I want"** (suena brusco), decí **"I would like"** o **"I'd like"**:

- ❌ I want a coffee. _(suena rudo)_
- ✅ I'd like a coffee, please. _(suena educado y natural)_

---

### 🔑 El truco de CAN para pedir todo

`Can` es tu comodín universal:

- **Can I have...?** → Para pedir algo en tienda/restaurante
- **Can you help me?** → Para pedir ayuda
- **Can you speak slowly?** → Para pedir que hablen más despacio
- **Can we...?** → Para sugerir algo

---

### ⚡ El truco de los conectores para sonar fluido

Estos conectores hacen que tu inglés suene más natural y fluido:

| Conector          | Uso               | Ejemplo                          |
| ----------------- | ----------------- | -------------------------------- |
| **Also / too**    | también           | I like coffee. Also, I like tea. |
| **But**           | pero              | I want to go, but I'm tired.     |
| **So**            | entonces/así que  | It's late, so I'm going home.    |
| **Because**       | porque            | I'm happy because I passed.      |
| **And**           | y                 | I work and study.                |
| **Actually**      | en realidad       | Actually, I don't agree.         |
| **I mean...**     | o sea...          | I mean, it's not that easy.      |
| **By the way...** | a propósito...    | By the way, did you call?        |
| **Anyway...**     | de todos modos... | Anyway, let's move on.           |

---

### 🗣️ Frases para GANAR TIEMPO mientras pensás

En cualquier idioma necesitamos segundos para pensar. Usá estas frases para no quedarte en silencio:

```
Well...                     → Bueno...
Let me think...             → Déjame pensar...
That's a good question...   → Esa es una buena pregunta...
Hmm, how can I say this...  → Hmm, cómo puedo decir esto...
I mean...                   → O sea...
You know...                 → Sabés...
Actually...                 → En realidad...
```

---

### 📱 Los 5 mejores hábitos para aprender inglés rápido

1. **Cambiá el idioma de tu celular a inglés** — Es la exposición diaria más fácil que existe.
2. **Mirá series con subtítulos en inglés** (no en español) — Empezá con algo que ya viste en tu idioma.
3. **Hablá solo en voz alta** — Practicá describir lo que hacés: "I'm making coffee. I'm going to the store."
4. **Aprendé 5 palabras nuevas por día** — No más. La consistencia gana a la intensidad.
5. **No traduzcas, asociá** — En vez de "cat = gato", pensá directamente en la imagen de un cat.

---

### 🎓 La regla de oro: El "Círculo de Exposición"

```
INPUT (absorber) → PROCESSAR → OUTPUT (producir)
```

- **Input:** Escuchar y leer inglés real (podcasts, series, música)
- **Procesar:** Entender el contexto, no traducir cada palabra
- **Output:** Hablar y escribir aunque cometas errores

> **El mayor error es esperar estar "listo" para hablar. Se aprende hablando, no estudiando.**
