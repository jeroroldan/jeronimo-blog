# Guía: Herramientas y Programas Esenciales 🛠️

> 💡 **En esta guía aprenderás**: Cómo instalar programas de forma segura, dominar el correo electrónico, elegir entre suites de oficina y configurar herramientas profesionales para diseño, programación y juegos.

---

## Parte 1: Instalación Segura de Programas

Existen dos formas principales de instalar software en Windows:
1. **Microsoft Store**: La opción más segura; Microsoft verifica cada app.
2. **Navegador Web**: Descargar instaladores (`.exe` o `.msi`). 

> [!CAUTION]
> Descarga software **siempre** desde el sitio web oficial del desarrollador. Evita sitios de "descargas gratuitas" de terceros, ya que suelen incluir malware o publicidad no deseada.

---

## Parte 2: El Mundo en tu Navegador (Google Chrome)

Google Chrome es el navegador más usado. Para dominarlo:
- **Pestañas**: Usa `Ctrl + T` para abrir una nueva.
- **Favoritos**: Haz clic en la estrella para guardar sitios importantes.
- **Historial**: `Ctrl + H` para ver qué sitios visitaste recientemente.

### Gmail: Tu Puerta al Mundo Digital ✉️
- **Creación Segura**: Usa una contraseña única (no la misma que en Facebook).
- **Protección**: Activa la **Verificación en Dos Pasos (2FA)**. Esto requiere tu móvil para entrar, haciendo casi imposible que te hackeen.

## Parte 3: Productividad y Creación

### 3.1 Word vs Google Docs: ¿Cuál elegir?
| Característica | Microsoft Word | Google Docs |
| :--- | :--- | :--- |
| **Costo** | Pago (Suscripción/Licencia). | Gratuito (necesitas cuenta Google). |
| **Conexión** | Funciona offline perfectamente. | Enfocado en la nube (online). |
| **Colaboración** | Limitada. | En tiempo real con varios usuarios. |

### 3.2 Diseño Gráfico: Photoshop vs Illustrator
- **Photoshop**: Basado en píxeles. Ideal para **retocar fotos** y crear composiciones realistas.
- **Illustrator**: Basado en vectores. Ideal para **logos** e ilustraciones que no pierden calidad al ampliarse.

## Parte 4: Herramientas Especializadas

### Programación (VS Code) 💻
Si quieres empezar a crear tus propias apps, **Visual Studio Code** es el estándar. Es ligero, potente y tiene miles de extensiones para cualquier lenguaje.

### Gaming: ¿Mi PC corre este juego? 🎮
Para jugar necesitas revisar los **Requisitos del Sistema**:
- **Mínimos**: Lo básico para que el juego abra (puede ir lento).
- **Recomendados**: La configuración para disfrutar el juego con buenos gráficos y fluidez.

## Conclusión

Tu computadora es tan potente como los programas que tiene instalados. Elegir las herramientas correctas y configurarlas con seguridad es vital para cualquier tarea.
