## Sección 1: Introducción al Trading

### Qué Es el Trading

Trading es comprar y vender instrumentos financieros con el objetivo de obtener bénéfices. A diferencia de invertir a largo plazo, el trader busca ganancias en períodos cortos.

### Categorías de Trading

| Categoría            | Temporalidad    | Enfoque               |
| -------------------- | --------------- | --------------------- |
| **Day Trading**      | Minutos a horas | Operaciones en el día |
| **Swing Trading**    | Días a semanas  | Capturar swings       |
| **Position Trading** | Semanas a meses | Tendencias largas     |

## Sección 4: Indicadores Técnicos

### Medias Móviles (Moving Averages)

| Indicador   | Uso                              |
| ----------- | -------------------------------- |
| **SMA 50**  | Tendencia media                  |
| **SMA 200** | Tendencia larga                  |
| **EMA**     | Más sensible a cambios recientes |

**Crossover**:

```
Precio cruza SMA50 sobre SMA200 → BUY
Precio cruza SMA50 bajo SMA200 → SELL
```

### RSI (Relative Strength Index)

```
RSI (14)
├── >70: Sobrecomprado (posible venta)
├── <30: Sobrevendido (posible compra)
└── 50: Neutral
```

### MACD (Moving Average Convergence Divergence)

| Componente                     | Señal               |
| ------------------------------ | ------------------- |
| **Línea MACD** cruza **Señal** | Cambio de tendencia |
| **Divergencia** con precio     | Señal de inversión  |
| **Cero line**                  | Confirmación        |

### Bandas de Bollinger

```
         ┃ banda superior ┃
    ─────┃───────────────┃───── precio
         ┃   banda media  ┃ (SMA 20)
    ─────┃���─��────────────┃───── precio
```

Uso:

- Precio cerca de banda superior → posible venta
- Precio cerca de banda inferior → posible compra
- Estirar bands → volatilidad alta

## Sección 5: Estrategia y Psicología

### Gestión de Riesgo (CRÍTICO)

> **💡 Concepto Clave**: Sobrevive primero. Ganar después.

**Regla de Oro**: Nunca arriesgues más del 2% por operación.

### Tamaño de Posición

```python
# Calcular tamaño de posición
account_size = 1000  # Tu capital
risk_percent = 0.02  # 2% máximo
entry_price = 1.1000
stop_loss = 1.0950

risk_amount = account_size * risk_percent
position_size = risk_amount / (entry_price - stop_loss)
```

### Apalancamiento

| Apalancamiento | Riesgo Real  |
| -------------- | ------------ |
| 1:1            | 1% = 1%      |
| 1:10           | 1% = 10%     |
| 1:100          | 1% = 100% ⚠️ |

> **⚠️ WARNING**: El apalancamiento amplifica ganancias Y pérdidas.

### Backtesting

```
Pasos:
1. Define estrategia clara
2. Aplica a datos históricos
3. Mide resultados
4. Ajusta si es necesario
5. Repite hasta consistente
```

**Métricas a revisar**:

- Win rate (>50%)
- Reward/Risk ratio (>1.5)
- Drawdown máximo (<20%)
- Expectativa positiva

### Psicología del Trading

| Emoción         | Problema           | Solución                   |
| --------------- | ------------------ | -------------------------- |
| **Miedo**       | Salir temprano     | Stop loss fijo             |
| **Avidez**      | Sobreoperar        | Límite diario              |
| **Esperanza**   | Mantener perdieron | Regla de 3 pérdidas = stop |
| **Frustración** | Operar por rabia   | Descansar después de racha |

**La Regla del Trading**:

```
Plan → Ejecutar → Revisar → Mejorar
   ↑                              │
   └──────────────────────────────┘
```

## Sección 6: Cripto

### Qué Son las Criptomonedas

Activos digitales descentralizados. Ningún banco central las controla.

### exchange Principales

| exchange     | Característica              |
| ------------ | --------------------------- |
| **Binance**  | Mayor volumen, muchos pares |
| **Coinbase** | Fácil para principiantes    |
| **Bybit**    | Buen para derivados         |

### Tips para Cripto

1. **Usa stop-loss siempre** - Volatilidad extrema
2. **No inviertes más de lo que puedes perder** - Pueden caer 99%
3. **Diversifica** - No pongas todo en un proyecto
4. **Estudia el proyecto** - No sigas solo el hype

## Errores Comunes

### ❌ Error 1: "Operar sin stop-loss"

**Mejor**: Siempre define tu pérdida máxima.

### ❌ Error 2: "Apalancamiento alto"

**Mejor**: Empieza con 1:1 o 1:2.

### ❌ Error 3: "No llevar registro"

**Mejor**: Every operación documentada.

### ❌ Error 4: "Operar por emoción"

**Mejor**: Si estás alterado, no operes.

### ❌ Error 5: "No practicar en demo"

**Mejor**: Demo hasta consistente.
