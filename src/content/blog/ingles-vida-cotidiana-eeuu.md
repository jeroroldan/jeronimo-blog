# Inglés para la Vida Cotidiana en Estados Unidos

Trabajar en Estados Unidos no significa vivir solo dentro de reuniones y código. También tenés que abrir una cuenta bancaria, contratar internet, hablar con el médico, entender una factura, alquilar un departamento, pedir comida, manejar transporte y resolver problemas diarios.

Estas conversaciones suelen ser más simples que una entrevista técnica, pero pueden generar ansiedad porque pasan en tiempo real, con acentos diferentes y presión social.

Esta guía te da frases prácticas para manejar la vida diaria en inglés.

---

## Objetivo de esta guía

Al terminar esta guía vas a poder:

- Abrir una cuenta bancaria.
- Contratar celular e internet.
- Hablar con servicios públicos.
- Pedir ayuda en tiendas y restaurantes.
- Manejar citas médicas básicas.
- Entender términos de alquiler.
- Comunicarte en transporte.
- Resolver problemas comunes.
- Pedir aclaraciones sin vergüenza.

## En el banco

Abrir una cuenta bancaria suele requerir documentos, dirección, número de teléfono y a veces historial crediticio.

### Frases útiles

```
I'd like to open a checking account.
I'd like to open a savings account.
What documents do I need?
Is there a minimum balance?
Are there monthly fees?
Can I open an account online?
How do I order a debit card?
How do I set up direct deposit?
How do I transfer money?
```

### Diálogo ejemplo

```
Bank employee: How can I help you today?
You: Hi, I'd like to open a checking account.
Bank employee: Sure. Do you have your ID and proof of address?
You: Yes, I have my passport and a utility bill.
Bank employee: Great. Let's get started.
```

Fee - comisión
Minimum balance - saldo mínimo
Direct deposit - depósito directo

## Llamar por teléfono

Muchas personas evitan llamadas en inglés. La clave es tener frases preparadas.

### Inicio de llamada

```
Hi, I'm calling about my account.
Hi, I need help with my internet service.
Hi, I'm calling to schedule an appointment.
```

### Si no entendés

```
Sorry, could you repeat that?
Could you spell that, please?
Could you say your name again?

```

### Pedir hablar con alguien

```

```

```

I appreciate it.
Have a good day.
```

## Pedir ayuda cuando no entendés

Esta es la habilidad más importante.

```
I'm sorry, I'm still learning English.
Could you speak slowly, please?
Could you repeat that?
Can you write it down?

## Números, fechas y dinero

En Estados Unidos, los números aparecen todo el tiempo: direcciones, teléfonos, precios, fechas, horarios.

### Fechas

Estados Unidos usa normalmente:

```
Month/Day/Year
```

Ejemplo:

```
06/19/2026 = June 19, 2026
```

Esto puede confundir porque en muchos países se usa día/mes/año.

### Dinero

```
$1.99 = one dollar and ninety-nine cents
$20 = twenty dollars
$1,500 = fifteen hundred dollars
$1,500.50 = fifteen hundred dollars and fifty cents
```

### Direcciones

```
425 Main Street
Apartment 3B
New York, NY 10001
```

Frases:

```
What's your address?
What's your apartment number?
What's your zip code?
```

## Errores comunes

### Error 1: Decir "I don't know" ante cualquier duda

Mejor:

```
I'm not sure. Could you explain it to me?
```

### Error 2: No pedir que escriban información importante

Si es una dirección, nombre de medicamento o número de cuenta, pedí que lo escriban.

```
Could you write that down for me, please?
```

### Error 3: Aceptar sin entender

Si no entendés una factura o contrato, preguntá.

```
Can you explain this fee?
What does this charge mean?
```

### Error 4: Confundir "ticket"

Ticket puede ser boleto o multa.

```
bus ticket - boleto de bus
parking ticket - multa de estacionamiento
```

```
