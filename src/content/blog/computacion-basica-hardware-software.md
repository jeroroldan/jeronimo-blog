# Guía: Hardware y Software desde Cero 💻

> 💡 **En esta guía aprenderás**: Los componentes esenciales de una computadora, cómo "piensa" en binario y los criterios clave para elegir tu próximo equipo según el uso que le vayas a dar.

---

## Parte 1: El Cerebro Detrás del Gráfico (Binario)

Las computadoras no entienden palabras ni imágenes directamente; todo lo traducen a **Binario**: un sistema de unos y ceros (bits).

```javascript
// Representación visual simplificada
1 = Encendido (Pasa corriente)
0 = Apagado (No pasa corriente)
```

**La escala de datos:**
- **Bit**: La unidad mínima (0 o 1).
- **Byte**: 8 bits (representa un carácter).
- **Kilobyte (KB)**: ~1,000 bytes.
- **Megabyte (MB)**: ~1,000,000 bytes.
- **Gigabyte (GB)**: ~1,000,000,000 bytes.

---

## Parte 2: Anatomía de una Computadora (Hardware)

El hardware es todo lo que puedes "patear" (lo físico). Cada parte tiene una misión crítica.

| Componente | Función Principal | Analogía |
| :--- | :--- | :--- |
| **Procesador (CPU)** | Realiza todos los cálculos y operaciones. | El cerebro. |
| **Memoria RAM** | Almacena datos temporales para acceso rápido. | La mesa de trabajo. |
| **Disco Duro (SSD/HDD)** | Guarda tus archivos de forma permanente. | El archivador/estante. |
| **Placa Madre** | Conecta todos los componentes entre sí. | El sistema nervioso. |

> [!TIP]
> Si quieres una computadora rápida, prioriza siempre un disco **SSD** sobre un HDD. La diferencia de velocidad es abismal.

---

## Parte 3: Guía para Comprar tu Computadora 🛒

No necesitas la PC más cara, necesitas la que se adapte a tu vida.

### 3.1 Según tu Uso:
- **Uso General/Oficina**: Mínimo 8GB RAM, Procesador Core i3/Ryzen 3.
- **Diseño/Programación**: Mínimo 16GB RAM, Procesador Core i5/i7 o Ryzen 5/7.
- **Gaming/Video 4K**: 16GB+ RAM, Tarjeta Gráfica dedicada (Nvidia/AMD).

### 3.2 Puertos y Pantalla:
- **Puertos**: Asegúrate de tener al menos 3 USB y 1 HDMI. El USB-C es el estándar del futuro.
- **Pantalla**: Busca resolución **Full HD (1920x1080)** para no cansar la vista.

---

## Parte 4: ¿Qué es el Software? El Alma del Equipo

El software son las instrucciones que le dicen al hardware qué hacer.

1. **Software de Sistema**: El Sistema Operativo (Windows, macOS, Linux). Es la base.
2. **Software de Aplicación**: Programas para tareas específicas (Chrome, Word, WhatsApp).
3. **Software de Desarrollo**: Herramientas para crear otros programas (VS Code).
