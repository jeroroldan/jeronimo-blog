## Parte 1: Fundamentos y Análisis Técnico

### 1.1 Acción del Precio (Price Action)

> **💡 Concepto Clave**: El precio lo dice todo. No necesitas indicadores.

#### Cómo Leer Velas Japonesas

```
Partes de una vela:
        Alto (máx)
         │
    Cuerpo │ Mecha superior
    (abrir├─│
    cerrar)│ │
         │ Mecha inferior
         │
    Bajo (mín)
```

| Tipo de Vela       | Qué Indica         |
| ------------------ | ------------------ |
| **Verde/larga**    | Presión compradora |
| **Roja/larga**     | Presión vendedora  |
| **Mecha larga**    | Rechazo de precio  |
| **Cuerpo pequeño** | Indecisión         |

### 1.2 Estructura del Mercado

```
    ZONA DE RESISTENCIA
           ╱╲
          ╱  ╲
         ╱    ╲_______ R1 (resistencia)
        ╱
       ╱    .     .
      ╱   .         .
 _____╱_____________ .___________
  .  .         .
  │         .
  │L (soporte)
  │
    ZONA DE SOPORTE
```

#### Identificar Soportes y Resistencias

1. **Múltiples tocados**: Más veces tocadas = más fuerte la zona
2. **Volumen**: Alto volumen = zona más relevante
3. **Reacción clara**:反弹 nítida = zona válida

### 1.3 Patrones de Velas

| **Martillo**       | Reversión alcista   | Alta     |

## Parte 2: Herramientas Avanzadas

### 2.1 Retrocesos de Fibonacci

```
         100%
          │
    ╱╲    │
   ╱  ╲   │
  ╱    ╲  │
 ╱      ╲ │
╱        ╲│
─────────── 0%
     ▲
  61.8% (golden ratio)
```

**Níveis críticos**: 23.6%, 38.2%, 50%, 61.8%, 78.6%

#### Cómo Usar Fibonacci

```
1. Identifica un movimiento (A→B)
2. Espera retrace hasta nivel FI
3. Busca señal de reversión
4. Entra en la dirección del movimiento original
```

### 2.2 Medias Móviles

| Tipo        | Período Común | Uso                 |
| ----------- | ------------- | ------------------- |
| **SMA 50**  | Corto plazo   | Tendencia reciente  |
| **SMA 200** | Largo plazo   | Tendencia principal |
| **EMA**     | Variable      | Más sensible        |

#### Cruces (Crossovers)

```
    SMA50 cruza sobre SMA200 = SEÑAL COMPRA ✓
    SMA50 cruza bajo SMA200 = SEÑAL VENTA ✗
```

### 2.3 Ondas de Elliott

> **🔍 Para Curiosos**: Teoría que identifica patrones recurrentes en los mercados.

```
    Onda 1: Impulso inicial
    Onda 2: Recesión (no pasa de 1)
    Onda 3: Generalmente la más larga
    Onda 4: Recesión (no pasa de 3)
    Onda 5: Impulso final

    ABC = Corrección posterior
```

**Reglas de oro**:

- Onda 3 NUNCA puede ser la più corta
- Onda 4 NO puede entrar en territorio de Onda 1
- Las ondas se subdividen en patrones menores

## Parte 3: Estrategias de Trading

### 3.1 Day Trading

Operar dentro del mismo día. Cierra antes de que cierre el mercado.

**Requisitos**:

- Temporalidad: 5M, 15M, 1H
- Precisión en entradas
- Gestión estricta de stop-loss
- No operar en news

### 3.2 Scalping en Criptomonedas

Operar en temporales muy bajas (1M-5M) buscando pequeños movimientos.

```
Estrategia breakout scalping:
1. Identifica rango estrecho
2. Espera ruptura con volumen
3. Entrada rápida
4. Stop ajust: 0.5-1%
5. Salida: 1.5-3%
```

### 3.3 Forex

Parés de monedas. Alto apalancamiento disponible.

**Pares principales**:

- EUR/USD (majors)
- GBP/USD
- USD/JPY

> **⚠️ Importante**: Alto apalancamiento = alto riesgo. Empieza con cuenta pequeña.

### 3.4 Fractalidad

> **💡 Concepto Clave**: Mismos patrones se repiten en todas las temporales.

```
        4H (dirección)
          │_______
          │       │ 1H (confirmación)
    _______│_______│______ 15M (entrada)
    │                 │
____│_________________│___ 1M (timing)
```

**workflow**:

1. **Temporalidad alta**: Identifica dirección
2. **Temporalidad media**: Confirma estructura
3. **Temporalidad baja**: Busca punto de entrada

## Parte 4: Tipos de Órdenes

| Orden           | Cuándo Ejecuta        | Uso                 |
| --------------- | --------------------- | ------------------- |
| **Market**      | Inmediato             | Entrada rápida      |

```
Ejemplo de orden completa:

├── Take-Profit: 1.2420 (75 pips)
└── Tamaño: 1 lote

Risk/Reward: 75/35 = 2.14 ✓
```

## Parte 5: Gestión de Riesgo

### La Regla Más Important

> **⚠️ Regla de Oro**: Nunca arriesgues más del 2% por operación.

### Cómo Calcular Tamaño de Posición

```python
# Tu capital: $10,000
# Riesgo máximo: 2%
# Riesgo en dólares: $200
# Stop-loss en pips: 30
# Valor por pip (EUR/USD): $10

Tamaño = $200 / (30 × $10) = 0.66 lotes
```

### Reward-to-Risk Ratio

| Ratio       | Evaluación     |
| ----------- | -------------- |
| < 1:1       | ❌ Evitar      |
| 1:1 a 1:15  | ⚠️ Riesga alto |
| 1:1.5 a 1:2 | ✅ Aceptable   |
| > 1:2       | ✅✅ Ideal     |

### Drawdown Máximo

```
No permitido: >10% del capital
Si llegas a 10%:
1. Para de operar
2. Revisa qué salió mal
3. Vuelve a operar solo cuando tengas confianza
```

## Parte 6: Profesionalización del Trader

### 6.1 Plan de Trading

```
Plan escrito debe incluir:
├── Mercados que operas
├── Horarios de operativa
├── Setup de entrada específico
├── Tamaño de posición
├── Stop-loss fijo
├── Take-profit fijo
└── Reglas de gestión emocional
```

### 6.2 Backtesting

```
Proceso de backtesting:
1. Define estrategia clara
2. Aplica a datos históricos
3. Mide resultados (mínimo 100 operaciones)
4. Analiza win rate y avg risk/reward
5. Ajusta si es necesario
6. Forward test en demo
```

**Métricas importantes**:

- Win rate (>50%)
- Avg risk/reward (>1.5)
- Drawdown máximo (<20%)
- Expectativa positiva: (Win% × Avg Gain) - (Loss% × Avg Loss) > 0

### 6.3 Trading Journal

Every operación debe registr:

```
| Fecha | Par | Entry | SL | TP | Resultado | Notas |
|-------|-----|-------|----|----|---------|-------|
| 15/04 | EUR/USD | 1.0850 | 1.0830 | 1.0900 | +$500 | Setup cumpla |
```

## Parte 7: Psicología del Trader

### Errores Psicológicos Comunes

| Error                        | Causa               | Solución               |
| ---------------------------- | ------------------- | ---------------------- |
| **Overtrading**              | Necesidad de acción | Plan con límite diario |
| **Revenge trading**          | Querer recuperar    | 3 pérdidas = para      |
| **Holding perdedores**       | Esperanza           | Stop-loss es sagrado   |
| **Tomar ganancias temprano** | Miedo               | Usa trailing stop      |
| **Ignorar el plan**          | Disciplina          | Registra y revisa      |

### La Mentalidad Correcta

```
✓ enfocarse en el proceso, no en el resultado
✓ Aceptar pérdidas como parte del juego
✓ Celebrar pérdidas Small con stop-hit
✓ Cada operación es independiente
✓ El mercado siempre tiene razón
```

### Señales de Alarma

> **⚠️ Criterios para parar**:

- 3 pérdidas seguidas
- Sentirte frustrado
- Operar por aburrimiento
- "Solo una más"

## Parte 8: Estafas y Mentiras del Trading

### Estafas Comunes

| Estafa              | Descripción               | Señal   |
| ------------------- | ------------------------- | ------- |
| **Bots garantía**   | "Gana 100%/día"           | Mentira |
| **Señales pagadas** | "100% precisión"          | Mentira |
| **CFDs complejos**  | Productos incomprensibles | Evitar  |
| **Esquemas Ponzi**  | Ganancias de otros        | ilegal  |
| **Cursos mágicos**  | "Sin esfuerzo"            | Farsa   |

### La Industria de Cursos

> **🔍 Análisis crítico**:

- Muchos cursos enseñan lo básico gratuito
- Pocos enseñan gestión real de risco
- El trading rentable requiere años de práctica
- No existe Holy Grail

### признаков de un curso legítimo

- ✅ Garantiza gestión de risgo
- ✅ Ensea a operar en demo primero
- ✅ No promete ganancias fáciles
- ✅ Incluye educación en riesgo
